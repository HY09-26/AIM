# AIM: Adversarial Information Masking for Cross-Modal Evaluation of Explanation Methods

This repository contains the audio and image experiment code for the **AIM** framework — a faithfulness evaluation benchmark for post-hoc XAI methods. AIM introduces two multi-domain masking strategies:

- **mdROAD** (multi-domain ROAD): in-distribution imputation using neighbourhood interpolation (spectrograms / images) and Multipoint Fractional Brownian Bridge noise (waveforms), preserving local signal structure while removing class-relevant information.
- **mdAR** (multi-domain Adversarial Robustness): replaces selected features with adversarially-perturbed counterparts generated by Projected Gradient Descent (PGD), testing whether explanations align with adversarial sensitivity.

Both strategies are applied under the standard **MoRF** (Most Relevant First) and **LeRF** (Least Relevant First) masking schedules, and evaluated with three area-based metrics: **AOC**, **ABC**, and **AUC**.

## Background

### Faithfulness Evaluation: MoRF and LeRF

A faithful attribution method should assign high scores to features that actually drive the model's prediction. AIM tests this by progressively masking features in importance order and measuring how accuracy degrades:

- **MoRF** (Most Relevant First): mask the top-scored features first → accuracy should drop sharply.
- **LeRF** (Least Relevant First): mask the bottom-scored features first → accuracy should remain high initially.

### Masking Strategies

| Strategy | Description | Modality |
|---|---|---|
| **Zero** | Replace masked features with zero (baseline) | All |
| **mdROAD** | In-distribution imputation (neighbour interp. / MFBB noise) | All |
| **mdAR** (PGD) | Adversarial imputation via Projected Gradient Descent | All |

### Evaluation Metrics

Let $\text{Acc}(x_k^M)$ / $\text{Acc}(x_k^L)$ be model accuracy after masking $k\%$ of features under MoRF / LeRF, and $|C|$ be the number of classes:

$$\text{AOC} = \frac{1}{K+1}\sum_{k=0}^{K} \frac{\text{Acc}(x_0) - \text{Acc}(x_k^M)}{\text{Acc}(x_0) - \frac{1}{|C|}}$$

$$\text{ABC} = \frac{1}{K+1}\sum_{k=0}^{K} \frac{\text{Acc}(x_k^L) - \text{Acc}(x_k^M)}{\text{Acc}(x_0) - \frac{1}{|C|}}$$

$$\text{AUC} = \frac{1}{K+1}\sum_{k=0}^{K} \frac{\text{Acc}(x_k^L) - \frac{1}{|C|}}{\text{Acc}(x_0) - \frac{1}{|C|}}$$

Higher values → explanation method more faithfully identifies the features that drive predictions.

**Spearman rank correlation** between MoRF and LeRF rankings (up to 50% masking ratio) measures framework consistency: a higher positive correlation indicates the two orderings are compatible, while a negative correlation suggests ranking inconsistency.

## Audio Experiments

### Datasets

| Dataset | Classes | Task | Representation |
|---|---|---|---|
| [AudioMNIST](https://github.com/soerenab/AudioMNIST) | 10 | Spoken digit recognition | Waveform + Spectrogram |
| [ESC-50](https://github.com/karolpiczak/ESC-50) | 50 | Environmental sound classification | Waveform + Spectrogram |
| [MSoS](https://zenodo.org/record/3553021) | 5 | Environmental sound classification | Waveform + Spectrogram |

ESC-50 uses 5-fold cross-validation; results are averaged across folds.

### Models

| Model | Input | Architecture |
|---|---|---|
| **AudioNet** | Waveform (B, 1, T) | 1-D CNN |
| **Res1dNet31** | Waveform (B, T) | 1-D ResNet |
| **AlexNet_Audio** | Spectrogram (B, 1, F, T) | 2-D CNN adapted for audio |
| **CNN14** | Log-mel (B, 1, mel, time) | 14-layer CNN (AudioSet pre-trained) |

### Attribution Methods

| Method | Code | Signed / Abs |
|---|---|---|
| Gradient (GD) | `gradient` / `gradient_abs` | Both |
| Gradient × Input (GI) | `gradinput` / `gradinput_abs` | Both |
| SmoothGrad (SG) | `smoothgrad` / `smoothgrad_abs` | Both |
| SmoothGrad² (SG²) | `smoothgrad_sq` | Abs |
| VarGrad (VG) | `vargrad` | Abs |
| Integrated Gradients (IG) | `integrad` / `integrad_abs` | Both |
| GradCAM | `gradcam` | Abs |
| GradCAM++ | `gradcampp` | Abs |
| Random | `random` | — |

### Running Audio Experiments

**Install dependencies:**
```bash
cd audio
pip install -r requirements.txt
```

**Waveform — rank-based MoRF/LeRF:**
```bash
python experiment/test_waveform.py \
  --dataset audiomnist \        # audiomnist | esc50 | msos
  --model audionet \            # audionet | res1dnet31
  --mask_type pgd \             # zero | pgd | road
  --expl_method gradient_abs \
  --n_steps 20
```

**Waveform — interval-based MoRF/LeRF:**
```bash
python experiment/test_waveform_interval.py \
  --dataset esc50 \
  --model res1dnet31 \
  --fold 1 \
  --mask_type road \
  --expl_method smoothgrad_abs
```

**Spectrogram / log-mel MoRF/LeRF:**
```bash
python experiment/test_spectrogram.py \
  --dataset msos \              # audiomnist | esc50 | msos
  --model cnn14 \               # alexnet | cnn14
  --mask_type pgd \
  --expl_method gradcam
```

**Compute AOC / ABC / AUC:**
```bash
python experiment/area.py \
  --root audio/ \
  --datasets audiomnist esc50 msos \
  --models audionet res1dnet31 \
  --masks zero pgd road
```

**Spearman consistency:**
```bash
python experiment/spearman.py \
  --root audio/ \
  --datasets audiomnist esc50 msos \
  --models audionet res1dnet31 alexnet cnn14 \
  --masks zero pgd road \
  --out spearman_audio.csv
```

**Plot MoRF/LeRF curves:**
```bash
python experiment/plot_morf_lerf.py \
  --datasets audiomnist esc50 \
  --models audionet \
  --masks pgd road
```

## Image Experiments

### Datasets

| Dataset | Classes | Task |
|---|---|---|
| [Oxford-IIIT Pet](https://www.robots.ox.ac.uk/~vgg/data/pets/) | 37 | Fine-grained cat/dog breed classification |
| [ImageNet](https://www.image-net.org/) | 1000 | Large-scale object recognition |
| [Brain Tumor MRI](https://www.kaggle.com/datasets/sartajbhuvaji/brain-tumor-classification-mri) | 4 | Medical image classification |

ImageNet and Oxford-IIIT Pet experiments use a fixed random subset of 500 samples for reproducibility.

### Models

| Model | Architecture | Weights |
|---|---|---|
| **ResNet-50** | Residual network | ImageNet pre-trained |
| **EfficientNet-B0** | Compound-scaled CNN | ImageNet pre-trained |
| **RepVGG-B0** | Re-parameterised VGG-style | ImageNet pre-trained (ImageNet); fine-tuned (others) |

Brain Tumor MRI uses grayscale images converted to 3-channel inputs.

### Attribution Methods

| Method | Code |
|---|---|
| Gradient (GD) | `gradient` |
| SmoothGrad (SG) | `smoothgrad` |
| GradCAM | `gradcam` |
| GradCAM++ | `gradcampp` |
| Random | `random` |

For image models, saliency maps are aggregated across RGB channels using absolute values.

### Running Image Experiments

**Install dependencies:**
```bash
cd image
pip install -r requirements.txt
```

**MoRF/LeRF evaluation:**
```bash
python experiment/test_image.py \
  --dataset oxford_pet \        # brain_mri | imagenet | oxford_pet
  --model resnet_50 \           # resnet_50 | efficientnet_b0 | repvgg_b0
  --mask_type pgd \             # zero | pgd | road
  --expl_method gradcam \
  --n_steps 20
```

**Compute AOC / ABC / AUC:**
```bash
python experiment/area.py \
  --root image/ \
  --datasets brain_mri imagenet oxford_pet \
  --models resnet_50 efficientnet_b0 repvgg_b0 \
  --masks zero pgd road
```

**Spearman consistency:**
```bash
python experiment/spearman.py \
  --root image/ \
  --datasets brain_mri imagenet oxford_pet \
  --models resnet_50 efficientnet_b0 repvgg_b0 \
  --masks zero pgd road \
  --out spearman_image.csv
```

**Plot MoRF/LeRF curves:**
```bash
python experiment/plot_morf_lerf.py \
  --datasets imagenet oxford_pet \
  --models resnet_50 \
  --masks pgd road
```

## Data and Checkpoint Setup

Preprocessed data and model checkpoints should be placed under the following paths (created automatically by the training scripts):

```
audio/experiment_utils/
├── data/
│   ├── audiomnist/preprocessed_data/
│   ├── ESC50/preprocessed_data/
│   └── MSoS/preprocessed_data/
└── checkpoints/
    ├── audiomnist/<model>/best_model.pth
    ├── esc50/<model>/fold_<N>/best_model.pth
    └── msos/<model>/best_model.pth

image/experiment_utils/
├── data/
│   ├── brain_mri/
│   ├── imagenet/
│   └── oxford_pet/
└── checkpoints/
    ├── brain_mri/<model>/best_model.pth
    ├── imagenet/<model>/best_model.pth
    └── oxford_pet/<model>/best_model.pth
```

Pre-computed saliency maps (`.npy`) for each (dataset, model, method) combination are stored under:
```
audio/expl_<dataset>/<model>/[fold_<N>/]<method>.npy
image/expl_image_<dataset>/<model>/<method>.npy
```
